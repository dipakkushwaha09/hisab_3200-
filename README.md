# खर्च का विवरण

## कुल राशि प्राप्त:
- ₹3200 — मनोज वाला खेत का हिसाब

## खर्च:
1. ₹720 — गाड़ी धोने का खर्च
2. ₹1000 — मम्मी को बाज़ार के सामान के लिए दिए
3. ₹1000 — वकील "शनि" को ब्याज दिया (अक्टूबर महीने वाला)
4. ₹1000 — ननै को दिए

---

## कुल खर्च:
₹720 + ₹1000 + ₹1000 + ₹1000 = ₹3720

## बाकी/घाटा:
₹3200 (प्राप्त) - ₹3720 (खर्च) = **₹520 का घाटा**
